W. Вставка символов
===================

Ограничение времени

1 секунда

Ограничение памяти

64Mb

Ввод

стандартный ввод или input.txt

Вывод

стандартный вывод или output.txt

Дана строка. Получите новую строку, вставив между двумя символами исходной строки символ \*. Выведите полученную строку.

Формат ввода
------------

Вводится строка.

Формат вывода
-------------

Выведите ответ на задачу.

### Пример 1

Ввод

Вывод

Python

P\*y\*t\*h\*o\*n

### Пример 2

Ввод

Вывод

Hello

H\*e\*l\*l\*o

### Пример 3

Ввод

Вывод

A

A
