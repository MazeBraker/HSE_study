Коля пишет класс "Дерево". Узел дерева может хранить целое число, а также знает о своём родителе и о своих потомках. У узла
            есть функция AddChild для добавления потомка с заданным числовым значением, а также функция Print для красивой печати поддерева
            начиная с этого узла. 
         Вот что получилось у Коли: https://clck.ru/FNSkV. 
      
Использоваться этот класс будет примерно так: https://clck.ru/FNSnA. Однако эта работающая на первый взгляд тестовая программа падает, если её собрать с адресным санитайзером. Исправьте код
         класса TreeNode, чтобы решить эту проблему.
      

Примечания

Сдайте в систему только код класса TreeNode без функции main. Подключите все необходимые для вашей реализации библиотеки.
