Дан шаблонный класс MathVector<T>, представляющий из себя математический вектор с элементами типа T (не путать с std::vector!). Код класса расположен здесь: https://clck.ru/AP4wf. Вам требуется исправить ошибки в коде этого класса (если они есть) и дописать операторы += и + для сложения векторов. Считайте, что складываться друг с другом всегда будут только векторы одинакового размера.

Примечания

В вашем решении должен быть только код класса и не должно быть функции main. При проверке наша программа test.cpp будет использовать ваш класс MathVector<T> и протестирует его.
