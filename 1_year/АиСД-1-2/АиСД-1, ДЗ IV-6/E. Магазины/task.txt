

E. Магазины


Ограничение времени
1 секунда


Ограничение памяти
10Mb


Ввод
стандартный ввод или input.txt


Вывод
стандартный вывод или output.txt





Фирма A продаёт книги в N городах. В K городах у фирмы A есть свой склад и магазин, и поэтому в них нет проблемы с тем, чтобы доставить товар вовремя и в нужном
            объёме, а в остальных N - K городах у фирмы A есть только магазин, и туда приходится подвозить книги из городов, в которых у фирмы есть склады. 
         Города соединены сетью платных автобанов, стоимость использования которых Pij не зависит от количества перевозимого груза. Также известно, что любой из складов фирмы A способен обеспечить товаром любое
         количество магазинов, и между любыми двумя городами проходит не больше одного автобана. Движение по всем автобанам возможно
         в обе стороны. 
      
За какую минимальную стоимость пользования дорогами фирма A может организовать доставку книг все свои магазины?

Формат ввода

Первая строка содержит через пробел три числа: 
            
N (1 ≤ N ≤ 105) - число городов, 
               
M (1 ≤ M ≤ min(N(N-1)/2, 106)) - число автобанов, соединяющих эти города, 
               
K (1 ≤ K ≤ N) - максимальный номер города, в котором у фирмы A есть склад: во всех городах 1 ≤ i ≤ K есть склад, во всех городах K < i ≤ N есть магазин, но склада нет. 
               

Следующие 2 * M строк содержат отсортированный по паре городов список автобанов. Каждая строка содержит через пробел три числа: 
         
i (1 ≤ i ≤ N) - начальный город автобана, 
            
j (1 ≤ j ≤ N) - конечный город автобана, 
            
Pij (0 ≤ Pij ≤ 103) - стоимость перевозки груза по этому автобану. 
            


Каждый автобан указан два раза - для прямого порядка городов и для обратного.

Формат вывода

Вывод должен состоять из одного числа - минимальной стоимости пользования дорожной сетью, при которой фирма A может обеспечить
            книгами все свои магазины.

Примечания

Следует считать, что каждый сегмент дорожной сети (i, j) обеспечен отдельным грузовиком достаточной вместимости для перевозки партии груза любого объёма, его стоимость не зависит
            от объёма груза, перевозимого этим грузовиком, и уже учтена в Pij.
