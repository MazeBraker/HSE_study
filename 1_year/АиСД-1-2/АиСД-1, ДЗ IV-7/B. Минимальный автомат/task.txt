

B. Минимальный автомат


Ограничение времени
1 секунда


Ограничение памяти
64Mb


Ввод
стандартный ввод или input.txt


Вывод
стандартный вывод или output.txt





Дан конечный автомат с n состояниями. Какое минимальное число состояний может быть в автомате, распознающем такой же язык?
         
Формат ввода

Первая строка содержит два числа:  — число состояний автомата и размер алфавита соответственно. 
         Следующие n строк содержат по l чисел каждая - j-ое число в i-ой строке задает номер состояния, в которое автомат переходит из состояния i, прочитав символ j. 
      
Следующая строка содержит одно число от 0 до n - 1 - номер начального состояния. 
      
Следующая строка содержит одно число k,  - количество терминальных состояний. 
      
Следующая строка содержит k чисел от 0 до n - 1 - номера терминальных состояний.
      

Формат вывода

Одно число - число состояний в минимальном автомате, эквивалентном данному.
Пример

Вывод
