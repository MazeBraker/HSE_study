C. Эквивалентные автоматы

Ограничение времени 1 секунда
Ограничение памяти 64Mb
Ввод стандартный ввод или input.txt
Вывод стандартный вывод или output.txt

Требуется определить, эквивалентны ли заданные автоматы, т.е. распознают ли они один и тот же язык.

Формат ввода

Входной файл содержит последовательное описание двух автоматов с одним и тем же алфавитом. Каждый автомат описывается следующим образом:

Первая строка описания содержит два числа: — число состояний автомата и размер алфавита соответственно.

Следующие n строк содержат по l чисел каждая: j-ое число в i-ой строке задает номер состояния, в которое автомат переходит из состояния i, прочитав символ j.

Следующая строка содержит одно число от 0 до n - 1 — номер начального состояния.

Следующая строка содержит одно число k, — количество терминальных состояний.

Следующая строка содержит k чисел от 0 до n - 1 — номера терминальных состояний.

Формат вывода

Выведите 0, если автоматы не эквивалентны, и 1, если эквивалентны.
