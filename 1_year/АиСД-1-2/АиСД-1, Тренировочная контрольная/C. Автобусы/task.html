C. Автобусы

Ограничение времени 0.2 секунды
Ограничение памяти 6Mb
Ввод стандартный ввод или input.txt
Вывод стандартный вывод или output.txt

Автобусы прибывают на остановку каждую минуту. В течение N минут приехало N автобусов, причем в i-ую минуту прибыл автобус с номером Ai. Так получилось, что для каждого номера рейса, проходящего через эту остановку, автобус с таким номером проезжал не менее двух раз.

Определите максимальное время между появлениями автобусов с одинаковыми номерами.

Формат ввода

Первая строка содержит целое число N (2 ≤ N ≤ 105) минут.

Вторая строка содержит N целых чисел Ai (1 ≤ Ai ≤ 105) — номера автобусов, прибывших на остановку в течение этих N минут. Каждый номер встречается не менее двух раз.

Формат вывода

Выведите одно целое число — наибольшее время в минутах между последовательным появлением автобусов с одинаковыми номерами.

Пример 1

Ввод Вывод 
7
23 5 3 3 23 5 23

4

Пример 2

Ввод Вывод 
5
2 2 1 2 1

2
