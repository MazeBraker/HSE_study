

C. Минимальная отсутствующая строка


Ограничение времени
2 секунды


Ограничение памяти
256Mb


Ввод
стандартный ввод или input.txt


Вывод
стандартный вывод или output.txt





Дан текст S, состоящий из N символов. В тексте встречаются только первые 10 строчных латинских букв (от a до j).
         Требуется найти строку минимальной длины, не содержащуюся в тексте. Если подходящих строк несколько, требуется вывести лексикографически
         минимальную (первую по алфавиту).
      

Формат ввода

Первая строка содержит целое число N (1 ≤ N ≤ 106) — длину текста S.
         Вторая строка содержит текст S, состоящий из N строчных латинских букв от a до j.
      

Формат вывода

Выведите минимальную строку, отсутствующую в тексте.
Пример 1



Ввод
Вывод




3
aab

c




Пример 2



Ввод
Вывод




22
aabacbbcadaeafagahaiaj

bd




Примечания

Результат — достаточно короткая строка. Сложите в бор все подстроки текста достаточной длины, а затем найдите ответ с помощью обхода
            этого бора.
