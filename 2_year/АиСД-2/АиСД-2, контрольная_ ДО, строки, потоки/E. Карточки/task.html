E. Карточки

Ограничение времени 2 секунды
Ограничение памяти 64Mb
Ввод стандартный ввод или input.txt
Вывод стандартный вывод или output.txt

Дан набор из N карточек. На одной стороне i-й карточки написана буква Ai, на другой — буква Bi.

Определите, можно ли из этих карточек сложить слово S. Каждую карточку можно использовать не более одного раза.

Формат ввода

Первая строка содержит целое число N (1 ≤ N ≤ 100) — количество карточек.

Следующие N строк описывают карточки. Каждая из них содержит строчные латинские буквы Ai и Bi — символы, записанные на лицевой и оборотной стороне карточки.

Следующая строка содержит слово S (1 ≤ |S| ≤ 100), состоящее из строчных латинских букв, которое требуется сложить из карточек.

Формат вывода

Выведите YES, если из карточек возможно составить слово S. В противном случае выведите NO.

Пример 1

Ввод Вывод 
5
a o
b c
c d
e d
c m
code

YES

Пример 2

Ввод Вывод 
8
a g
p p
r r
g o
m p
a z
o f
x p
program

NO

Пример 3

Ввод Вывод 
1
a a
aa

NO
